Use *on* as a preposition with:

  - Menus
  - Tabs
  - Taskbar, toolbar, ruler, and desktop
  - Disks, in the sense of a program being on a disk
  - Networks
  - Hardware platforms
  - The web

In COM programming, an interface is implemented *on* an object.
